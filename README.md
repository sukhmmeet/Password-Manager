# 🔐 Password Manager App

A simple and secure Android password manager built using Firebase. This app helps users store and manage their credentials safely with a clean and minimal UI.

## ✨ Features
- 🔑 User Authentication (Firebase)
- 🔒 Secure password storage
- ⚡ Auto-login with saved session
- 📱 Simple and clean UI

## 🛠 Tech Stack
- Kotlin
- Firebase Authentication
- Firebase Firestore / Realtime DB
- Jetpack Compose
